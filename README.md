# AegisKit

**Secure. Local. Modular.**

A dependency-less, privacy-first web toolkit designed for secure file manipulation, generation, and conversion. All operations execute locally in the browser or securely on the host server.

## Core Utilities (Planned)

- Image Processing & Conversion
- Client-Side PDF Manipulation
- Offline QR Code Matrix Generation
- Custom URL Shortening routing

## Stack

- Next.js (App Router)
- Pure CSS Modules
- Zero 3rd-Party Tracking
